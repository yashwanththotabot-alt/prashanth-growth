## Your Mission

### Step 1: Install Git
```bash
# Check if installed
git --version

# If not installed, download from git-scm.com
```

### Step 2: Set Up Git
```bash
git config --global user.name "Prashanth"
git config --global user.email "your@email.com"
```

### Step 3: Your First Repository
```bash
# Create a folder
mkdir my-first-code
cd my-first-code

# Initialize Git
git init

# Create a file
echo "Hello, real code!" > hello.txt

# Add and commit
git add hello.txt
git commit -m "My first commit - from Pega to Python!"
```
